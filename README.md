## Research

### Virtual Gaze-Driven Keyboard

**In collaboration with Dr. George K. Thiruvathukal**

[Read the full paper](https://doi.org/10.6084/m9.figshare.31883932)

I designed and implemented a hands-free, webcam-based virtual keyboard that enables text input through eye movement and blinks. The system uses MediaPipe FaceMesh and Eye Aspect Ratio detection, with real-time iris tracking, calibration controls, keyboard-boundary logic, predictive text, and dwell heatmaps.

The project explores how browser-based computer vision can produce low-cost accessibility tools without specialized eye-tracking hardware.
